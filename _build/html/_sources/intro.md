# 开卷小语

这是一个用来记录个人阅读的小站。

选书偶有目的性，却不侧重特定主题，不遵循任何规律；全凭当时兴趣所致，想到什么就找什么来读。读书不追求速度，时常三天打鱼两天晒网，只能任凭待读书单野蛮生长。

本人记忆力欠佳，读过之书如过眼云烟，很快就在大脑里踪迹难寻。作此小站，权当借助外力开发"第二大脑"，弥补先天不足。
